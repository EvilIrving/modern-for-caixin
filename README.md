# Modern for Caixin · 下载与发布

> 让财新（`*.caixin.com`）更清爽、长文更好读，像一份安静的现代报刊。

这是 **Modern for Caixin** 浏览器扩展的**分发站点**——下载页与安装包都在这里。
源码不在本仓库公开。

- **下载页**：<https://evilirving.github.io/modern-for-caixin/>
- **安装包**：见 [Releases](https://github.com/EvilIrving/modern-for-caixin/releases)，提供 Chrome / Edge / Firefox 三个平台的解压即装版本。
- **隐私政策**：<https://evilirving.github.io/modern-for-caixin/privacy.html>
- **问题反馈**：[Issues](https://github.com/EvilIrving/modern-for-caixin/issues)

## 安装（手动加载）

1. 在 [Releases](https://github.com/EvilIrving/modern-for-caixin/releases) 下载对应平台的 zip。
2. 解压。
3. 浏览器打开扩展管理页，开启「开发者模式」，「加载已解压的扩展程序」选择解压目录。

## 说明

Modern for Caixin 是第三方、非官方项目，与财新传媒无关联，也未获得财新官方背书。
扩展只重排你当前有权访问的页面内容，不解锁付费正文、不破坏登录态、不上传阅读内容。
