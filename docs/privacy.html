隐私政策

Modern for Caixin · 最后更新：2026-06-28

Modern for Caixin 在你的浏览器本地工作，主要读取当前打开的财新页面 DOM，用来重新排版页面。少数列表区块会请求财新的匿名公开 JSON 数据补全，不携带 cookie。设置、阅读清单、历史足迹、高亮和 Pro 状态只保存在浏览器本地。扩展不收集、出售、传输或分析你的个人数据。

我们不收集什么

  • 不上传浏览历史、阅读记录或访问频率。
  • 不上传文章正文、标题、评论、页面截图或页面 HTML。
  • 不读取密码，不保存或传输财新的账号、cookie、订阅信息或支付信息。
  • 不使用遥测、埋点、广告 SDK、第三方统计脚本或远程日志。

扩展在本地做什么

扩展仅在匹配 *.caixin.com 的页面上运行。它读取当前页面已经渲染出来的 DOM，提取标题、正文、列表、图片和页面结构信息，然后在浏览器本地生成阅读界面。少数页面的空列表区块会请求财新的匿名公开 JSON 接口补全；这些请求使用 credentials: "omit"，不携带财新 cookie。

权限说明

扩展需要访问 *://*.caixin.com/*，因为它只服务财新网页，需要在这些页面上读取 DOM 并注入本地阅读界面。这个权限不用于访问其他网站。扩展还需要 storage 权限，用于在浏览器本地保存设置、阅读清单、历史足迹、高亮和 Pro 状态，不上传。扩展会本地读取昵称 cookie 来镜像登录显示，但不会保存、修改或传输 cookie。

付费内容与登录态

Modern for Caixin 不解锁付费正文，不绕过登录、订阅或付费墙，也不修改财新的账号体系。你能看到什么内容，仍由你与财新之间的登录状态和订阅权限决定。

Pro 授权码

GitHub 自安装版可能包含 Pro 预览能力。正式授权码验签尚未开放；开放后会在浏览器本地校验，不发送到服务器。商店免费版不包含站外付费解锁流程。

第三方关系

Modern for Caixin 是第三方、非官方项目，与财新传媒无关联，也未获得财新官方背书。请在遵守财新网服务条款的前提下使用。

联系方式

如果你发现隐私或安全问题，请通过 GitHub issue 联系：github.com/EvilIrving/modern-for-caixin/issues。
